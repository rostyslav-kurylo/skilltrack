# SkillTrack

Project for skill tracking with

## Getting Started

### Install dependencies

```bash
npm install
```

### Run the app locally

```bash
npm run start
```

The app will be available at http://localhost:3000

### Build for production

```bash
npm run build
```

The build output will be in the `dist/` folder.

### Run tests

```bash
npm run test
```

Tests use Jest and React Testing Library

### Project structure

- `src/` — source code
- `src/components/` — React components
- `src/pages/` — app pages
- `src/store/` — Zustand store
- `src/i18n.ts` — i18next setup and translations
- `public/` — static files

### Technologies

- React, TypeScript, Zustand, Material UI, Chart.js, i18next, Jest, React Testing Library
